## <b id="0"> ❓ 어떤 프로젝트인가요?(프로젝트의 목적)</b>
캡스톤 디자인에서 사용했던 Firebase 플랫폼의 서버를 활용하여 웹 서비스를 구현해 보고 싶어, 

html, jquery, Firebase를 활용한 당근 마켓 클론 코딩 강의를 수강하였고 

위에서 배운 기술을 활용하여 음식 관련 커뮤니티 서비스를 구현해 보았습니다.
